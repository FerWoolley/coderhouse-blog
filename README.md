# Coderhouse - Blog Técnico: Mentalidad de Crecimiento y Comunicación

Este repositorio contiene el código fuente y el diseño del blog técnico realizado para la materia **Mentalidad de Crecimiento y Comunicación en Entornos Digitales** de la Diplomatura de Desarrollo Full Stack en **Coderhouse**.

El blog aborda un desafío real de desarrollo web: la depuración de errores de **CORS** y la persistencia de **cookies de sesión HTTP-Only** en un despliegue de producción con dominios cruzados (React en Vercel, Express/Node.js en Render).

## Contenido del Repositorio
* `index.html`: Estructura semántica del blog técnico con el caso de estudio completo.
* `styles.css`: Estilo visual premium con soporte responsive, degradados modernos y micro-interacciones.
* `entregable_coderhouse.md`: Documento entregable con el checklist completo requerido por la cátedra.

## Desafío Abordado
El caso documentado describe paso a paso:
1. **Contexto**: Entorno MERN (React, Node, Express, MongoDB) desplegado.
2. **Problema**: Bloqueo de peticiones CORS en producción y pérdida de credenciales de sesión.
3. **Acciones**: Diagnóstico post-mortem constructivo, reconfiguración del backend Express y Axios en el frontend.
4. **Aprendizajes**: Lecciones clave sobre la gestión de entornos de despliegue.

## Reflexión sobre Feedback Radicalmente Sincero
Al final de la entrada se detalla una reflexión práctica sobre cómo el equipo resolvió un cuello de botella de despliegue mediante el modelo de **Feedback Radicalmente Sincero (Radical Candor)**, combinando un alto *Afecto Personal* con un *Desafío Directo* para resolver el error constructivamente y sin fricciones destructivas.
